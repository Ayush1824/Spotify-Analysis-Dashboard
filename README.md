# 🎵 Spotify Music Trends Analysis Dashboard

An end-to-end data analytics project analysing 50,000+ Spotify tracks across 10+ years to uncover music trends, top artists, genre patterns, and audio feature insights — built using Power BI, SQL, and Excel.

## 🎯 Project Objective

To analyse Spotify's music dataset and answer key business questions:
- Which artists and genres dominate across different years?
- How have audio features (energy, tempo, danceability) evolved over time?
- What patterns define a "popular" track?

---

## 🔍 Key Insights

- 📈 **Top genres** shifted significantly between 2010–2023, with pop and hip-hop dominating recent years
- ⚡ **Energy & danceability** scores increased consistently after 2015
- 🎤 Identified the **top 10 artists** by track count and average popularity score
- 🎼 Tracks with **higher valence (positivity)** tend to chart more frequently in recent years
- ⏱️ Average track duration has **decreased by ~30 seconds** over the last decade

## 📌 Dashboard Features

- **4-page interactive report** with 8 dynamic slicers (year, genre, artist, mood)
- Audio feature trend lines across decades
- Top artist & genre leaderboards
- Mood-based track filtering (energetic, calm, happy, sad)
- Reduced manual filtering time from **20 minutes to under 30 seconds** per query

## 🚀 How to Use

1. Download `Spotify.pbix`
2. Open with **Microsoft Power BI Desktop** (free download at powerbi.microsoft.com)
3. Explore the interactive dashboard using slicers and filters
